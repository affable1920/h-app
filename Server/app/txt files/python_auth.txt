2. passlib[bcrypt]
-
pwd hashing
from passlib.context import CryptCOntext
pwd_context = CryptCOntext(
    schemes=["bcrypt"] // hashing alg to user
    deprecated="auto" //  auto-updates old hashes
    )

usage -
- hashing
pwd_context.hash(pwd: the plain pwd string)

- verifying
pwd_context.verify(plain_pwd, hashed_pwd)
